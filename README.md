# AzerbicaCliento
ArabicaCliento, но в попытках починить и модифицировать.

[Discord server](discord.gg/nTAzURC6h5)


## Требования
Что бы запустить это, вам нужен [MarseyLoader](https://github.com/AZERBAIJAN-TECH/SGLoader) или [SGloader](https://github.com/AZERBAIJAN-TECH/SGLoader-V2)

## Установка
1. Скачатйте последнюю версию [MarseyLoader](https://github.com/ValidHunters/Marseyloader/releases)/[SGloader](https://github.com/AZERBAIJAN-TECH/SGLoader-V2/releases)
2. Перейти в [релизы](https://github.com/AZERBAIJAN-TECH/AzerbicaCliento/releases).
3. Скачивайте последнюю версию AzerbicaCliento (.dll).
4. Поместите файл в папку Mods по пути `Marsey/Mods`.
5. Запустите MarseyLoader, перейдите в *Plugins*, и активируйте ArabicaCliento.
6. Enjoy!

## Билд
Если вы хотите создать чит для целей разработки, начните с его скачивания.

1. Склонируйте репозиторий:
    ```bash
    git clone --recurse-submodules https://github.com/AZERBAIJAN-TECH/AzerbicaCliento.git
    ```
2. Перейдите в директорию:
    ```bash
    cd AzerbicaCliento
    ```
3. Забилдите:
    ```bash
    dotnet build
    ```
4. После билда, азербика будет находтся в `ArabicaCliento/bin/Debug/net10.0/ArabicaCliento.dll`. Скопируйте в mods марсея и используйте.

## Функции

- **Melee Aimbot** 
- **Ranged Aimbot** 
- **Anti-Slip** 
- **Auto-Spin** 
- **Auto-Zoom** 
- **Friend-or-Foe System (Arabica Friend)**
- **ClickGui** - интерфейс, меню который открывается на F4
